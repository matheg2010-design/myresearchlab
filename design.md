# Design Style Guide - Scientific Research Platform

## Design Philosophy
- **Academic Professionalism**: Clean, trustworthy design that conveys expertise and reliability
- **Modern Minimalism**: Sophisticated simplicity with purposeful use of space and typography
- **Cultural Sensitivity**: Right-to-left (RTL) design optimized for Arabic content
- **Accessibility First**: High contrast ratios, clear navigation, and intuitive user flows

## Color Palette
- **Primary**: Deep Navy (#1a365d) - Professional, trustworthy, academic
- **Secondary**: Warm Gold (#d69e2e) - Highlighting, call-to-actions, accent elements
- **Tertiary**: Soft Gray (#f7fafc) - Backgrounds, subtle sections
- **Text**: Charcoal (#2d3748) - Primary text, high contrast
- **Success**: Emerald (#38a169) - Form submissions, positive feedback
- **Warning**: Amber (#d69e2e) - Alerts, important notices

## Typography
- **Display Font**: "Amiri" (Arabic-optimized serif) - For headings and important text
- **Body Font**: "Cairo" (Modern Arabic sans-serif) - For body text and UI elements
- **English Font**: "Inter" - For technical terms and mixed content
- **Hierarchy**: 
  - H1: 2.5rem, bold
  - H2: 2rem, semibold  
  - H3: 1.5rem, medium
  - Body: 1rem, regular
  - Small: 0.875rem

## Visual Language
- **Geometric Patterns**: Subtle Islamic geometric patterns as background elements
- **Scientific Imagery**: Abstract data visualization, research-themed illustrations
- **Professional Photography**: Academic settings, diverse researchers, modern laboratories
- **Icon System**: Consistent line-based icons with 2px stroke weight

## Visual Effects & Animations
- **Libraries Used**:
  - Anime.js: Smooth micro-interactions and page transitions
  - ECharts.js: Interactive data visualizations for statistical tools
  - Splide.js: Image carousels and content sliders
  - p5.js: Background particle effects and creative coding elements
  - Pixi.js: Advanced visual effects for hero sections

- **Animation Style**:
  - Subtle fade-ins and slide-ups on scroll
  - Smooth hover states with 3D transforms
  - Typewriter effects for key headings
  - Interactive data visualizations with smooth transitions

## Layout Principles
- **Grid System**: 12-column responsive grid with RTL support
- **Spacing**: Consistent 8px base unit system
- **Sections**: Clear visual hierarchy with generous whitespace
- **Navigation**: Fixed header with smooth scrolling to sections

## Component Styling
- **Cards**: Clean white backgrounds with subtle shadows and gold accent borders
- **Buttons**: Rounded corners, smooth hover animations, clear hierarchy
- **Forms**: Floating labels, real-time validation, progress indicators
- **Data Visualizations**: Consistent color scheme, interactive tooltips, responsive design
