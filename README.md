# منصة بحثي - دليل التركيب والاستخدام

## نظرة عامة

منصة بحثي هي منصة إلكترونية متخصصة في دعم الباحثين والطلاب في مجال البحث العلمي، توفر أدوات إحصائية، إرشادات، واستشارات متخصصة.

## المميزات

- **أدوات إحصائية تفاعلية**: معالج ذكي لاختيار الاختبار الإحصائي المناسب
- **دليل البحث العلمي**: إرشادات متكاملة لكتابة البحث وتوثيق المصادر
- **نظام استشارات**: نموذج طلب مساعدة مع إمكانية رفع الملفات
- **تصميم متجاوب**: يعمل على جميع الأجهزة والشاشات
- **واجهة عربية متكاملة**: دعم كامل للغة العربية مع تصميم RTL

## المتطلبات

### المتطلبات الأساسية
- Node.js (الإصدار 14 أو أحدث)
- MySQL (الإصدار 5.7 أو أحدث)
- npm (الإصدار 6 أو أحدث)

### المتطلبات الاختيارية
- بريد إلكتروني Gmail للإشعارات
- حساب PayPal للمدفوعات

## خطوات التركيب

### 1. تحميل الملفات
```bash
git clone https://github.com/yourusername/bahithi-platform.git
cd bahithi-platform
```

### 2. تثبيت التبعيات
```bash
npm install
```

### 3. إعداد قاعدة البيانات

#### إنشاء قاعدة البيانات
```sql
CREATE DATABASE bahithi_platform CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

#### إعداد ملف الإعدادات
انسخ ملف `.env.example` إلى `.env` وعدل الإعدادات:
```bash
cp .env.example .env
```

عدل ملف `.env` وأضف معلوماتك:
```env
# Database Configuration
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your-database-password
DB_NAME=bahithi_platform

# Email Configuration (اختياري)
EMAIL_USER=your-email@gmail.com
EMAIL_PASS=your-app-password

# Server Configuration
PORT=3000
NODE_ENV=development
```

### 4. تشغيل الخادم

#### في وضع التطوير:
```bash
npm run dev
```

#### في وضع الإنتاج:
```bash
npm start
```

### 5. فتح المتصفح
افتح المتصفح واذهب إلى:
```
http://localhost:3000
```

## الإعدادات المتقدمة

### إعداد البريد الإلكتروني
لتفعيل إشعارات البريد الإلكتروني:

1. قم بإنشاء كلمة مرور تطبيق في Gmail:
   - اذهب إلى إعدادات حسابك في Google
   - اختر "الأمان"
   - فعل "المصادقة الثنائية"
   - أنشئ "كلمة مرور تطبيق"

2. أضف البيانات إلى ملف `.env`:
```env
EMAIL_USER=your-email@gmail.com
EMAIL_PASS=your-app-password
```

### إعداد قاعدة البيانات على خادم خارجي
إذا كنت تستخدم خادم قواعد بيانات خارجي:
```env
DB_HOST=your-database-host.com
DB_USER=your-database-username
DB_PASSWORD=your-database-password
DB_NAME=bahithi_platform
```

## الاستخدام

### كمستخدم
1. تصفح الصفحات المختلفة من القائمة الرئيسية
2. استخدم معالج الاختبار الإحصائي لاختيار الاختبار المناسب
3. تصفح دليل البحث العلمي للحصول على الإرشادات
4. املأ نموذج طلب الاستشارة للحصول على دعم شخصي

### كمسؤول
- يمكنك الوصول إلى قاعدة البيانات لعرض طلبات الاستشارة
- تفعيل إشعارات البريد الإلكتروني لتلقي الطلبات الجديدة
- مراجعة الملفات المرفوعة في مجلد uploads/

## الأمان

### نصائح الأمان المهمة
1. **غيّر كلمات المرور الافتراضية**
2. **استخدم HTTPS في الإنتاج**
3. **حدث التبعيات بانتظام**:
   ```bash
   npm update
   ```
4. **أنشئ نسخ احتياطية من قاعدة البيانات**

### حماية الملفات المرفوعة
- يتم فحص نوع الملفات قبل الرفع
- الحد الأقصى لحجم الملف: 10MB
- الأنواع المسموح بها: PDF, Word, Excel, TXT, Images

## الصيانة

### تحديث المنصة
```bash
# تحديث التبعيات
npm update

# إعادة تشغيل الخادم
npm restart
```

### النسخ الاحتياطي
```bash
# نسخ احتياطي لقاعدة البيانات
mysqldump -u root -p bahithi_platform > backup.sql

# نسخ احتياطي للملفات
tar -czf backup.tar.gz public/ uploads/ server.js package.json
```
